# Restaurant ordering website
A simple restaurant website that allows to view the menu and place order. Orders are received and managed by resturant staff

## Features

### Customer
- View restaurant menu
- See food price
- Add items to order
- Remove items from order
- See total price
- Enter name, phone, location and other details
- Submit order

### Restaurant side
- View incoming order
- See ordered item + quantity
- See customer details
- Mark order as complete

## Technology used
- HTML
- CSS
- JAVASCRIPT

- Online database / backend
